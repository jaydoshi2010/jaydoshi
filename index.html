Automatic Article Summarization Web Application

  • Article Summarization refers to the technique of shortening long pieces of text. The intention is to create a coherent and fluent summary having only the main points outlined in the document. It is the process of finding the most important information from a document to produce an abridged version with all the important ideas.
  • Here in this application use of spaCy. It is a free, open-source library for advanced Natural Language Processing (NLP) in Python. spaCy provides the fastest and most accurate syntactic analysis of any NLP library released to date. It also offers access to larger word vectors that are easier to customize.

Auto Deployment using Jenkins

  • First of we have to build Two Linux EC2 instance on AWS. After We have to install Jenkins and install Tomcat server on this particular Instance. Jenkins is Continue integration tool. So, first we have to provide git URL whichever code is deploy on tomcat.
  • After git we have to set maven in Build. Where Maven is build tool. Build war file in workspace directory of the Jenkins and after we have to configure Post-Build Actions. where in Post-Build Actions we have to choose WAR/EAR files and Choose Tomcat Version Which is installed in Tomcat Ec2 Instance and also write Tomcat Server URL and We have to set Tomcat Credentials in Jenkins to connect with Tomcat Server.
  • Our Project Auto-deploy so, we have to Configure the Poll SCM (Source Code Manager) to regularly Checkout in SCM within 1 minute. To check the Any Modification is there or not. Any modification is there in the SCM then Jenkins automatically Start Build trigger and deploy this change in Tomcat Server.
